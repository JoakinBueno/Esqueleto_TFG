# Esqueleto_TFG
Este es mi proyecto intermodular que trata sobre una segmentación de Redes basado en el Modelo Purdue y mediante VLAN's.
He usado Cisco Packet Tracer para hacer una simulación configurando un Switch para añadir distintas VLANs y luego asignarlos a diferentes equipos.
Aqui dejo el archivo de cisco subido para que se pueda ver como esta todo configurado.
